Login Up

details of user who want to login
username (optional)
String username contains between 3 to 8 letters
example: sergio12
password (optional)
String password contains between 5 to 10 chars, at least one of them is a number and a special char format: password
example: sergio@124

PreviewRecipe Up

name (optional)
String name of the recipe
example: pizza
image (optional)
String URL for the recipe's image format: URL
example: https://GrandmaRecipes.com/pizza-image
AmountOfTime (optional)
Float The amount of time it takes for the entire preparation process format: float
example: 3.5
Popularity (optional)
Long amount of likes the recipes recieved format: int64
example: 30
Vegan (optional)
Boolean vegan or not
example: false
Gluten (optional)
Boolean Free Gluten Or Not
example: true
Watched (optional)
Boolean Indication if the user has already viewed the recipe
example: true
Saved (optional)
Boolean Indication if a user Save the recipe to favorites
example: true

SearchRecipe Up

schema for searching recipe
number (optional)
String number of recipes to return
example: 5
cuisine (optional)
String cuisine of recipes to return
example: italian
diet (optional)
String diet type of recipes to return
example: vegetarian
intolerance (optional)
String the return recipes must not contain ingredients that are not suitable for people with the intolerances entered.
example: gluten
